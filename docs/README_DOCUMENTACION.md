# Documentacion del proyecto

Este directorio contiene la documentacion fuente en Markdown:

- `01_documentacion_funcional.md`
- `02_documentacion_tecnica.md`
- `03_documentacion_despliegue.md`

- `output/pdf/01_documentacion_funcional.pdf`
- `output/pdf/02_documentacion_tecnica.pdf`
- `output/pdf/03_documentacion_despliegue.pdf`

Para regenerar los PDFs:

```powershell
python tools/generar_documentacion_pdf.py
```
